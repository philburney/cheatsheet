Html/Css Cheatsheet

Table Properties

Table Parts

Element Description
<table> The main table container
<thead> Contains the column Headings
<tbody> Contains the main cells
<tfoot> Contains the table Footer

Each table row must start with tr and then within that add td for the contents of each cell in the row. The exception is the header cells whose contents can be set with th.

Table Formatting

You can format tables by using the normal font properties such as text-align, font size and text decoration.
You can set borders but in order to make them collapse set the border-collapse to collapse in your css file.

Setting up your Html File

Html Parts

Element Description
<!DOCTYPE html> This is the declaration that it is HTML5 and should be the very first entry in the file
<html> tags This is the main container for your html file and must be the start and end of your file. You can add the language
<head> This contains heading information for your html. Items inside are not displayed "on the page". It should include the title and a link to any extenal css files
<body> Contains the main html elements which will be displayed on the page. May be broken down into sections, navs, headers, mains and footers amongst others

These can be set up quickly by either creating a template html file with these elements already present or if you are using visual studio then type html:5 and press tab. This will create a skeleton html file for you

Linking to an External CSS

In your <header>, use a <link> tag. Point to the css using href='file location' and set the rel='stylesheet'. You can link to multiple style sheets if needed.

Setting up your CSS

Resetting Defaults

It can be sensible to reset the browser defaults in your CSS file. This prevents unpredictable results as each browser has a different set of defaults that it works from. By setting your own defaults you can overide these from the start.

Consider setting the following global css defaults using the * selector

  • margin: 0;
  • padding: 0;
  • box-sizing:border-box;

Additionally, you may wish to set the border-collapse: collapse; for table elements
